## How To Use

#### Installation
```bash
  $ npm install
  $ npm run react-build
  $ npm run uglify
  $ npm run rollup
  $ npm run express-dev
```


#### API Reference
>Inside of the server file a config.js will need to be created.
```javascript
  module.exports = {
  API_ROUTE: 'https://app-hrsei-api.herokuapp.com/api/fec2/hr-den/',
  API_KEY: 'this is where your github token would go',
  };
```
